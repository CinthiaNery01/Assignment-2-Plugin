CreepyFace is a fun JavaScript Plugin that adds an interactive, eye-tracking effect to images, making them appear to "follow" the user's cursor or touch movement! This plugin makes your website more fun with just 13 lines of code.

CreepyFace Plugin Features

Creepyface JavaScript library is a unique and interactive tool that allows web developers to create an engaging and somewhat eerie user experience. This plugin tracks the user's cursor or touch movement, making an image appear to "follow" them with its eyes. This effect can be applied to various projects, adding an element of interactivity and surprise. Here are some key features of the Creepyface plugin:
Creepyface will automatically detect your image (thanks to the data-creepyface attribute) and make it look at the mouse or fingers depending on which device you are using.
You can add as many Creepyfaces as you want as long as they all have the data-creepyface attribute.

CreepyFace Plugin Use Cases

    Here are a few interesting use case examples for integrating Creepyface.
  • Interactive Websites & Landing Pages

    Creepyface can be used to make websites more engaging by giving characters or mascots a more dynamic presence. This can be particularly useful in:
    _Marketing pages that want to add a unique element to captivate visitors;
    _Portfolio sites where designers or developers want to showcase creative coding skills;
    _Horror or mystery-themed websites to enhance the eerie atmosphere.

  • Gamification in Web Experiences

    By integrating Creepyface into games or interactive storytelling experiences, developers can create:
    _Mini-games where users need to escape the gaze of a character;
    _Puzzles or escape room experiences where the eyes provide clues or react to user actions;
    _Engaging storytelling elements, such as a painting that "watches" the player.

  • E-commerce and Online Stores

    Online shops can leverage Creepyface to add a playful and memorable touch to their experience:
    _Animated shopping carts that follow users as they browse products;
    _Product pages with mascots that react as customers browse.
